Google Chrome / Firefox Network Har to CSV
This tool parses the content of HAR file from Network tab of Google Chrome Browser to CSV so that the network protocol can be opened with Excel for enhancing searching and filtering
In the network tab, just right click on any row and select Copy --> Copy all as HAR like following and paste the content into text area here
Or you can download the HAR file and then parse file directly
